# World Explorer

World Explorer is a Next.js project that allows users to explore countries around the world.

## Features

- App Router
- File-based routing
- Shared layout
- Dynamic routes
- Server components
- Client components
- Real API data fetching
- Static rendering and caching
- Dynamic rendering
- Search functionality
- Region filter interface
- Country comparison page
- Favorite countries using local storage
- Loading page
- Custom 404 page
- Dynamic country metadata

## API Used

REST Countries API

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
